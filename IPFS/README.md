# IPFS
* 安裝
  * MacOS 下載
    wget https://dist.ipfs.io/go-ipfs/v0.4.18/go-ipfs_v0.4.18_darwin-amd64.tar.gz
  * Linux 下載
    wget https://dist.ipfs.io/go-ipfs/v0.4.18/go-ipfs_v0.4.18_linux-amd64.tar.gz    
  * 解壓縮
    tar -zxvf go-ipfs_v0.4.17_linux-amd64.tar.gz
  * 切換工作路徑
    cd go-ipfs
  * 安裝 IPFS
    sudo ./install.sh
